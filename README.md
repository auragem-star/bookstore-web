# بوت واتساب لمكتبة الكتب الدراسية 📚🤖

هذا المشروع عبارة عن بوت واتساب ذكي يربط شيت جوجل يحتوي على تفاصيل الكتب الدراسية (الناشر، اسم الكتاب، السعر) مع حساب واتساب بيزنس للمكتبة. يقوم البوت بمساعدة الزبائن في معرفة أسعار الكتب في ثوانٍ معدودة وبخطوات بسيطة ومحددة.

## المميزات ✨
- **مربوط بشيت جوجل مباشرة:** يقرأ أسعار الكتب لحظياً (مع كاش لمدة 5 دقائق لتحسين الأداء وسرعة الرد).
- **عرض الكتب المتاحة فقط:** يستبعد تلقائياً الكتب التي لا يوجد سعر مسجل لها بجانبها في الشيت (الغير متوفرة حالياً).
- **قوائم تفاعلية (Interactive Lists):** تصفح سهل للصفوف الدراسية والكتب بضغطة زر.
- **دعم الترقيم والصفحات (Pagination):** بما أن صفوف الابتدائي تحتوي على أكثر من 10 كتب، يقوم البوت بتقسيم الكتب تلقائياً إلى صفحات (السابق ⬅️ والتالي ➡️) ليتناسب مع قيود واتساب (الحد الأقصى للقائمة هو 10 خيارات).

## هيكل محادثة البوت 💬
1. **الترحيب:** يرحب بالزبون ويطلب منه اختيار المرحلة الدراسية (الابتدائي وكي جي / الإعدادي والثانوي).
2. **الصف الدراسي:** يعرض قائمة بالصفوف التابعة للمرحلة التي اختارها الزبون.
3. **الكتاب:** يعرض الكتب المتوفرة للصف الدراسي مع إمكانية التنقل بين الصفحات.
4. **عرض السعر:** يعرض سعر الكتاب والناشر ويسأل الزبون إذا كان يرغب في البحث عن كتاب آخر.

## متطلبات التشغيل ⚙️
- **Node.js** (إصدار 18 أو أحدث)
- حساب **Meta for Developers** لإعداد WhatsApp Business Cloud API.
- رابط CSV لشيت جوجل (معد مسبقاً في ملف `.env`).

## طريقة التشغيل محلياً 💻

1. قم بتثبيت الاعتماديات:
   ```bash
   npm install
   ```

2. قم بإنشاء ملف `.env` واملأ البيانات الخاصة بك:
   ```env
   WHATSAPP_TOKEN=توكن_الوصول_الخاص_بـ_واتساب
   WHATSAPP_PHONE_ID=معرف_رقم_الهاتف_الخاص_بـ_واتساب
   WEBHOOK_VERIFY_TOKEN=توكن_التحقق_الذي_تختاره
   SHEET_CSV_URL=https://docs.google.com/spreadsheets/d/1dJMN6Bv9heYDzQHJfg1_PyUwcNmp0CR18H0rWBKouJ8/export?format=csv
   PORT=3000
   ```

3. قم بتشغيل السيرفر في وضع التطوير (الساخن):
   ```bash
   npm run dev
   ```

4. استخدم أداة مثل **ngrok** لعمل نفق للسيرفر المحلي ليكون متاحاً لـ Meta:
   ```bash
   ngrok http 3000
   ```
   استخدم الرابط الموفر من ngrok (مثل: `https://xxxx.ngrok-free.app/webhook`) كـ Webhook URL في إعدادات تطبيق Meta.
